== Description ==

The LemonPlugin was is a simple easy to use image slider that is added to the page using a shortcode.

There are several options to the shortcode such as:
	- Arrows
	- Dots
	- Images to show at a time
	- AutoPlay
	
This was developer by "Developer" Josh Fisher 
	

== Installation ==

This section describes how to install the LemonPlugin and get it working.

e.g.

1. Upload the LemonPlugin files to the `/wp-content/Plugins` directory, or install the LemonPlugin through the WordPress Plugin screen directly.
2. Activate the LemonPlugin through the 'Plugin' screen in WordPress
3. Use the Settings->Lemon Shortcode Settings screen to configure the LemonPlugin
4. Then configure your plugin and insert it into your desired page.
